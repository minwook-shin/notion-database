# Security Policy

## Supported Versions

| Version | Supported          | Notes                          |
|---------|--------------------|--------------------------------|
| 2.0.x   | :white_check_mark: | Active maintenance             |
| 1.4.x   | :warning:          | Security fixes only            |
| < 1.4.0 | :x:                | No longer supported            |

- **2.0.x** — 기능 추가, 버그 수정, 보안 패치 모두 진행합니다.
- **1.4.x** — 보안 취약점 패치만 제공하며, 기능 추가 및 일반 버그 수정은 진행하지 않습니다.
- **1.4.0 미만** — 지원이 종료되었습니다. 2.0 으로 업그레이드를 권장합니다.

## Reporting a Vulnerability

보안 취약점을 발견하신 경우 공개 이슈 트래커 대신 아래 방법으로 비공개 보고해 주세요.

1. GitHub **[Security Advisories](https://github.com/minwook-shin/notion-database/security/advisories/new)** 를 통해 비공개로 제출합니다.
2. 가능한 경우 재현 방법, 영향 범위, 제안하는 수정 방향을 함께 작성해 주세요.
3. 보고 접수 후 **72시간 이내** 에 초기 응답을 드립니다.
4. 취약점이 확인되면 패치 버전을 릴리스하고 보고자를 크레딧에 기재합니다 (원하는 경우).
